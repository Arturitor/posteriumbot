# Posterium — сервис для отложенного постинга и ведения Telegram-каналов

[![Website](https://img.shields.io/badge/Сайт-posterium.pro-34A853)](https://posterium.pro)

**Posterium** — современная платформа для отложенного постинга и удобного ведения Telegram-каналов.

Объединяет в себе **Telegram-бота**, **Mini App** и **полноценный веб-сайт**. Работай так, как тебе удобнее — с телефона или компьютера.

### 🔥 Главные особенности

#### Навигация по каналу (уникальная фича)
- Автоматическое создание **каталога постов** в удобной лёгкой Mini App-странице
- Один закреплённый навигационный пост с кнопкой «Навигация»
- Посты автоматически группируются по **потокам**
- Можно добавлять старые посты из канала (даже созданные без бота)
- Неограниченный объём страницы + красивый интерфейс, без обязательных ручных правок

#### Потоки (Рубрики)
- Отдельные ленты публикаций со **своим расписанием** и настройками
- Пресеты настроек: автоудаление, закрепление, защита контента, автоподпись и т.д. 
- Закрепление до 5 потоков в быстром доступе для планирования одним кликом
- Удобное разделение контента (новости, юмор, реклама, лонгриды и др.)

#### Массовый постинг
- Загружай до 10 файлов за раз — каждый файл становится отдельным постом
- До 200 постов за одну сессию
- Автоматическое распределение по свободным слотам расписания

#### Расписание и Постинг
- Возможно запланировать выход до 10 постов за одну минуту
- Индивидуальное расписание для каждого потока

#### Автоподписи
- До 10 автоподписей
- Авточередование или ручной выбор
- Привязка подписей к потокам
- Простой и продвинутый режим

#### Дополнительно
- Дубль и Репост в другие каналы (на уровне потока или отдельного поста)
- Inline-кнопки с выбором цвета и сохранением шаблонов
- Защита контента, автоповтор, автоудаление, спойлеры и многое другое
- Поддержка **16 языков** (русский + машинный перевод на другие языки, будет улучшаться)

### Платформы

- **Mini App** — удобный визуальный редактор, для мобильных устройств 
- **Сайт** posterium.pro — идеально для работы с компьютера
- **Чат-бот** — базовая версия

### Как начать

1. Перейди в бота → [@Posteriumbot](https://web.telegram.org/#@Posteriumbot)
2. Нажми **Начать** и создай персонального бота, затем подключи свои каналы
3. Создавай посты и настраивай потоки через чат, Mini App или сайт
